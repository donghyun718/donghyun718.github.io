Options / Data

1. data

object type

function type

  • methods는 key가 메소드 이름
  • 호출되면 함수 수행

computed 

new Vue({
    computed :{
        fullName: function () {
            return this.firstName + ' ' + this.lastName
        }
    } 
});
  • computed는 명시를 안하면 기본적으로 get임
  • computed는 key가 메소드 이름
  • 함수에 종속된 변수의 값(예시. firstName, lastName)이 바뀌면 수행

watch 

new Vue({
    watch: {
        firstName: function (val) {
            this.fullName = val + ' ' + this.lastName
        },
        lastName: function (val) {
            this.fullName = this.firstName + ' ' + val
        }
    }
});
  • key 값으로 지정한 변수의 값이 바뀌면 수행

computed vs watch 

<div id="app">
    <h1>{{ num }}</h1><br>
    {{ calculated }}
    <button @click="count --">-</button>
</div>

<script>
new Vue({
    el: "#app",
    data: {
        num: 3,
    },
    computed: {
        calculated: function () {
            if(this.num === 2) {
                  alert('num is 2')
            }
        }
    },
    watch: {
        count: function (newVal) {
            if(newVal === 0) {
                  alert('num is 0')
                this.num = 3
            }
        }
    }
})
</script>
  • 를 호출하는 부분이 없으면 num 값이 변경돼도 함수가 수행되지 않아 출력해줘야 한다.
