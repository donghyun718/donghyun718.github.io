기본적인 ML의 용어와 개념

1. ML

: 개발자가 일일이 정하지 않고 프로그램 자체가 데이터를 학습해서 무언가를 배우는 프로그램

2. Supervised Learning

: 레이블(결과값)들이 정해져있는 데이터로 학습

  • image labeling
  • email spam filter
  • predicting exam score

3. Unsupervised Learning

: 레이블이 정해져있지 않는 데이터로 스스로 학습.

4. Training data set

  • predicting exam score
    • score(0~100)는 범위가 넓어 regression
    • P/F로 나눈다면 binary classification
    • grade를 준다면 여러 레이블 중 나눈다면 multi-label classification

train_data -> regression -> result

Linear Regression

1. (Linear) Hypothesis

H(x) = W * x + b

2. Cost function

H(x) - y -> (H(x) - y)^2

cost function은 보통 차이를 제곱한다. 일정하게 차이를 양수로 표현하고 차이가 작을 때보다 클 때 패널티를 더 줄 수 있다.

cost function의 값을 최소화하는 W와 b의 값을 구하는 것이 목표

Linear Regression cost function 최소화

  • Gradient descent algorithm
    • minimize cost function
    • 경사도가 0일 때를 찾는 알고리즘
    • W값을 조금 바꾸면서 cost 값을 줄이는 과정을 반복
